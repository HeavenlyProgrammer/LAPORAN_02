  1. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web modern?
  2. Jelaskan konsep dan tujuan utama dari Responsive Web Design (RWD)!

JAWABAN

  1. Karena CSS pada pemrograman web modern merupakan bahasa stylesheet yang digunakan untuk mengontrol presentasi visual dan tata letak halaman web yang ditulis di dalam HTML, CSS juga membuat desain responsif, Konsistensi Visual, dan Kustomisasi tampilan HTML. Sedangkan Javascript dalam pemrograman web digunakan karena JS merupakan bahasa pemrograman yang membuat halaman web menjadi lebih dinamis dan interaktif, JS fungsi utama adalah menambahkan interaktivitas, dan membuat konten dinamis.

  2. Responsive Web Design (RWD) adalah sebuah pendekatan web yang berutjuan agar tampilan sebuah situs web dapat beradaptasi secara otomatis dengan berbagai ukuran layar dan perangkat yang digunakan oleh user. RWD intinya adalah membangun satu situs web yang fleksibel, bukan beberapa versi situs untuk perangkat yang berbeda. Tujuan prima dari RWD ini adalah untuk memberikan pengalaman pengguna atau UX (User Experience) yang optimal dan konsisten di semua jenis macam perangkat.

  3. Kombinasi antara CSS, JS dan RWD merupakan fondasi utama dalam menciptakan halaman web modern yang tidak hanya interaktif juga adaptif di berbagai macam perangkat. Sederhananya saja HTML adalah kerangka tulang, CSS adalah Fisik luarnya seperti pakaian, Javascript adalah saraf, otak, dan juga ototnya.

    Ketiganya saling melengkapi misalnya dalam contoh interface, contoh paling nyata adalah pada menu navigasi mobile (hamburger menu). Dalam kasus ini, RWD mendeteksi keterbatasan layar dan menginstruksikan untuk menyembunyikan menu navigasi panjang; CSS bertugas menyembunyikan elemen tersebut dan menyiapkan animasi sliding yang mulus; sementara JavaScript bertindak sebagai pemicu "saklar" yang mendeteksi sentuhan jari pengguna untuk membuka atau menutup menu tersebut. Tanpa JavaScript, interaksi tidak akan terjadi tanpa CSS, transisi akan terlihat kaku; dan tanpa RWD, tata letak akan berantakan. Kolaborasi inilah yang menghasilkan web yang tidak hanya adaptif terhadap ukuran layar, tetapi juga responsif terhadap perilaku manusia.
